# Daily Activity Log

A simple web application to log daily activities with options to filter, edit, delete, and export to PDF.

## Features

- Add new activities with time, date, description, duration, status, and notes.
- Edit existing activities.
- Delete activities.
- Filter activities by date.
- Export the activity log as a PDF document.

## Technologies Used

- **React**: For building the user interface.
- **Bootstrap**: For styling the application.
- **jsPDF**: For generating PDF reports.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, etc.)
- Internet connection (for loading external libraries)

### Installation

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd daily-activity-log
   ```

2. **Open the HTML File**:
   Open `index.html` in your preferred web browser. You can do this by double-clicking the file or dragging it into the browser window.

### Usage

1. **Adding an Activity**:
   - Fill in the details in the form on the left side.
   - Click "Add Activity" to save the activity.

2. **Editing an Activity**:
   - Click the pencil icon next to the activity you want to edit.
   - Modify the details and click "Update Activity".

3. **Deleting an Activity**:
   - Click the trash icon next to the activity you want to delete.

4. **Filtering Activities**:
   - Use the date filter to view activities for a specific date.
   - Click "Clear Filter" to remove the date filter.

5. **Exporting to PDF**:
   - Click the "Download PDF" button to export the list of activities to a PDF file.

## Customization

You can customize the application by modifying the following files:

- **index.html**: The main HTML file where the structure and layout are defined.
- **style**: Modify the CSS styles in the `<style>` section for different looks.
- **JavaScript Logic**: Customize the React components and functionality as needed.
